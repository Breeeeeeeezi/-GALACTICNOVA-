# Galactic Nova

Galactic Nova is a curated browser game website with a purple theme, glowing particles, background lofi music, and a clean pixel font.  

## Features
- Play a variety of curated and underrated browser games.
- Overlay iframe with fullscreen and "visit original site" buttons.
- Favorite games (future update possible).
- Search games by keywords (planned feature).
- Subtle floating particles for a galactic aesthetic.
- Calming lofi background music with volume control.

1. Click any game card to open the game overlay.

3. Adjust music volume with the slider at the bottom-right.
